# SAGE Lab Manual Submission Portal

A full-stack MERN web application for SAGE University Bhopal to manage lab manual submissions.

## Features

- **Student Role**: Upload and manage lab manuals based on academic year, semester, and subject.
- **Admin Role**: Faculty can review, approve, or reject submissions.
- **Authentication**: Secure login & signup using JWT.
- **File Management**: Upload PDF lab manuals.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB
- **Database**: MongoDB Atlas
- **Storage**: Local uploads / Cloudinary (TBD)

## Setup

1.  **Backend**:
    ```bash
    cd backend
    npm install
    npm start
    ```

2.  **Frontend**:
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
